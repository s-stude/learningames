Memory

The 15-puzzle is a sliding puzzle that consists of a frame of numbered square tiles in random order with one tile missing. The object of the puzzle is to place the tiles in order by making sliding moves that use the empty space. In this version of the game you do not just have to move the tiles, but you have to do it "in the blind". Let's go!
